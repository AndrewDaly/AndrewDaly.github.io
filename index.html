AutoHotKey is a powerful scripting language to automate scripts and create custom hotkeys.


Inspired by almogtavor's script (linked below) I created my own AHK static hands script written in AHK version 2

In my opinion, this type of shortkey script should be more widely adopted

Keyboard navigation should be more widespread for efficiency as well as accessibility reasons

My script centers hotkeys around 'j', one of the least used letters in the English language

'j' also sits on the homerow of most keyboards, this script repurposes it to be more useful
